# GAK Frontend

Frontend application for **GAK (Gyaan, Karma, Ahara)**, a student day assistant focused on academics, fitness, and nutrition.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Recharts

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Test

```sh
npm run test
```
